the type representing the minimax value

the type representing the contents of a potential board state

type t =
| Nodes of state * t list
* t option

the type of the game tree

val init_tree : State.GameState.t -> t

init_tree st creates the root of the game tree based on the gamestate st

val minimax : State.GameState.t -> int

minimax st is the minimax value for the board state st based on the material value of pieces left on the board for each player

val add_depth : t -> t

add_depth n is node n with all possible child nodes (potential game states) added

val next_move : State.GameState.t -> move

next_move st is the most profitable move to make based on minimax values of all potential moves
